😷 Face Mask Detection using Deep Learning

📌 Project Overview

This project focuses on building a Face Mask Detection system using Deep Learning techniques. The model is capable of classifying whether a person is wearing a mask or not based on image data.

It is a beginner-to-intermediate level project demonstrating the application of computer vision in real-world scenarios like public safety and health monitoring.

---

🚀 Features

- Classifies images into Mask and No Mask
- Built using Convolutional Neural Network (CNN)
- Image-based prediction system
- Simple and effective deep learning pipeline

---

🧠 Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/your-username/face-mask-detection.git
cd face-mask-detection

2. Create environment (recommended):

conda create -n ml_env python=3.10
conda activate ml_env

3. Install dependencies:

pip install -r requirements.txt

---

▶️ Run the Project

Open the ".ipynb" file using:

- Jupyter Notebook
- or Google Colab

Run all cells to train and test the model.

---

📊 Model Details

- Model Type: Convolutional Neural Network (CNN)
- Task: Binary Classification (Mask / No Mask)
- Input: Image
- Output: Prediction (Mask / No Mask)
